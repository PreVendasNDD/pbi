Bem-vindo ao NDD CARGO

Sistema completo de gestão para transportadoras. Importe seu arquivo Excel para carregar todos os módulos e funcionalidades do sistema.

Importante: O arquivo Excel deve conter as colunas "Aba", "Sub Aba" e "O que é / O que faz" conforme o modelo padrão.

Cada "Aba" se tornará um módulo principal no sistema, com suas "Sub Abas" como funcionalidades específicas.
